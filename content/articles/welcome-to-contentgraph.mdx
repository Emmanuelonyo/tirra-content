## Welcome!

This is your first content entry. It was automatically created when you set up ContentGraph.

You can:
- Edit this entry in the dashboard editor
- Create new entries via the dashboard or by pushing MDX files directly to this repo
- Import blog posts from Medium, Substack, Dev.to, and more using AI extraction

### How it works

ContentGraph treats this GitHub repository as a database. Every MDX file in the `content/` directory is a content entry. The frontmatter (the YAML between the `---` markers) defines the structured fields, and the body below is the rich content.

Changes you make in the dashboard are committed directly to this repo. Changes you push via Git are automatically picked up by the API within seconds.
